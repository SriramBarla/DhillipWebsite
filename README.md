# TownSquare (FBLA Project)

This repository hosts the TownSquare website.

## Live Site Source
- `index.html` is the website file used by GitHub Pages.

## Notes
- All businesses, users, and reviews shown in the app are fictional for educational/demo purposes.
- Data persistence is browser-based (`localStorage`) in the current implementation.

## Local Preview
1. Open `index.html` in your browser.
2. For best results, use a local static server if needed.
